# House-Price-Prediction-
I developed an interactive House Price Prediction web app using Streamlit and regression-based machine learning models. The app allows users to input property details and get instant price estimates. I enhanced the user experience by adding custom CSS for a modern and visually appealing interface, making the tool both functional and user-friendly.
